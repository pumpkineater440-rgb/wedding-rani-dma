
# Undangan Rani & Dimas - Ready to Deploy (Vite + React + Tailwind)

Ini paket proyek React + Tailwind yang sudah siap **di-deploy ke Vercel**. Paket ini dibuat agar mudah di-upload langsung dari HP Android atau desktop.

## Cara pakai cepat (di Vercel)
1. Unzip / ekstrak paket ini (opsional, Vercel bisa deploy langsung dari zip via GitHub upload atau manual).
2. Upload ke repository GitHub baru (rename repo sesuai keinginan).
3. Di Vercel: New Project -> Import Git Repository -> pilih repository -> Deploy.
4. Di Vercel, build command default (`npm run build`) dan output `dist` akan otomatis terdeteksi oleh Vercel.

## Menjalankan di lokal (opsional)
1. Install Node.js (v16+)
2. `npm install`
3. `npm run dev`
